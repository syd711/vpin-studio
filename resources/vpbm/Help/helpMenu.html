Help Menu

Virtual Pinball Backup Manager - managing your backups, one game at a time

  Help Menu Option

The Help menu option is used to open the Main help page in the default browser.

  Report Bug / Request Feature Menu Option

This menu option launches the Github New Issue page to allow bugs to be reported and new features requested.

  Change Log Menu Option

This menu option displays the change log.

  About Menu Option

The About menu option presents an About box describing the VPBM application.
